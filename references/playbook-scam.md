# Scam and Fraud Investigation Playbook

Work through this systematically when investigating potential scams:

| Investigation Layer | Technique | Red Flag Signals |
|---|---|---|
| 1. Domain analysis | WHOIS lookup: registration date, registrant info, registrar | Domain registered days/weeks ago, privacy-shielded WHOIS, cheap registrar, auto-generated domain name |
| 2. Website forensics | Archive.org history, technology stack analysis, SSL certificate details | No history before recent date, or history shows completely different previous business. Free SSL. Template site. |
| 3. Team verification | Reverse image search every team photo, search names in LinkedIn/academic databases | Stock photos, AI-generated faces, names that don't exist on any professional platform |
| 4. Content analysis | Check if website text is copied from legitimate companies, check for template patterns | Plagiarized "About Us" from real companies, generic text with only the company name swapped |
| 5. Template scam detection | Compare site structure/HTML with known scam templates, search for similar sites | Multiple identical sites with only names/logos changed — indicates a scam factory |
| 6. Financial claims verification | Compare promised returns with market benchmarks, regulatory warnings | "Guaranteed" returns, returns far above market average, no risk disclosure |
| 7. Regulatory check | Search SEC, CFTC, FCA, local financial regulators for warnings or registrations | Company not registered as required, or actively warned about by regulators |
| 8. Blockchain analysis (crypto) | Search wallet addresses on blockchain explorers (Etherscan, Blockchain.com) | Funds immediately transferred out, mixer usage, connections to known scam wallets |
| 9. Victim reports | Search "[company name] scam", "[company name] review", check complaint forums | Multiple victims reporting inability to withdraw funds, changing terms |
| 10. Legal entity verification | Check if the claimed company registration actually exists in the stated jurisdiction | Non-existent registration, registration in a different country than claimed |
